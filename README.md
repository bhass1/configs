### 10-evdev.conf
[Configuration file for Xorg X server.](https://www.x.org/releases/X11R7.6/doc/man/man5/xorg.conf.5.xhtml) Goes in the system's `xorg.conf.d` directory. On Ubuntu 20.04 it is: `/usr/share/X11/xorg.conf.d/`.
